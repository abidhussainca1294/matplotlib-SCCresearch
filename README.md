# matplotlib-SCCresearch -Pymaceuticals, Inc.

Overview: Animal study has been performed on 249 mice who were identified with SCC tumors. They received treatment with a range of drug regimens, over the course of 45 days. The tumor development was observed and measured.

Purpose: The purpose of this study is to compare the performance of Pymaceuticals’ drug of interest, Capomulin, against the other treatment regimens. A technical report consisting of Tables and figures are generated for the clinical study.

Input data: Input are stored within “data” folder.

Python dependencies used:  

* Matplotlib.pyplot
* Pandas
* Scipy.stats


The project has been broken down into the following tasks:

1. Data preparation.

2. Generation of summary statistics for drugs.

3. Creation of bar charts and pie charts.

4. Calculation of quartiles, find outliers, and creating a box plot.

5. Creating a line plot and a scatter plot.

6. Calculating correlation and regression.
